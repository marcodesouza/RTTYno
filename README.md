# RTTYno
Biblioteca RTTY no Arduino

###Testado com
- Arduino Nano
- Alto-falante 8ohm 0.25W ligado ao pino 8
- Biblioteca RTTY modificada
- decodificação com minimodem

```
minimodem --rx 50 --mark 2295 --space 2125 -8
```
